Problem Statement

Implement a customizable navigation map of all buildings on campus that would show both current location and a searchable destination. The map would allow for customization for special events such as open house. Ideally, this customization would be simple and easy to operate.

Students and campus visitors have to either rely on an outdated paper map, ask staff or other students for directions, or navigate by stickers on the floor during open houses. It is not easy, and results in many students and visitors being confused and not being able to navigate the campus efficiently.

  • Students becoming lost and late to classes
  • Extra staff is needed to help students navigate during the first few weeks of a new semester and special events
  • Visitors don't see all the campus has to offer and require a tour guide to be able to navigate and see the campus

Use existing ArcGIS Online data to create a navigation map, hosted in an ArcGIS web app such as Experience Builder or ArcGIS Indoors. Users will be able to see their location and the location of a searchable destination, with a route generated if time permits. Additionally, the web application will be configured to be easily updated and/or changed for special events.

The project utilizes a combination of ArcGIS Online, ArcGIS Experience Builder, and Leaflet technologies for mapping and interactive user experiences.
